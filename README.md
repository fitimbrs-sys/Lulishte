# 🌸 Lulishte - Premium Flower Shop

A beautiful, modern flower shop website with a soft green color theme, formal yet social-friendly design. Perfect for a boutique flower business.

## ✨ Features

- **Responsive Design**: Fully responsive on desktop, tablet, and mobile devices
- **Soft Green Theme**: Elegant palette featuring soft greens, sage, and cream colors
- **Product Gallery**: Showcase flower arrangements with ratings and pricing
- **Shopping Cart**: Add items to cart with local storage persistence
- **Newsletter Signup**: Email subscription form for customer engagement
- **Contact Form**: Customer inquiry and feedback form
- **Social Integration**: Social media links for shop connectivity
- **Smooth Animations**: Elegant transitions and scroll animations
- **Fast & Lightweight**: Minimal dependencies, vanilla JavaScript

## 🎯 Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-green: #2d5f4f;
    --light-green: #98b4a8;
    /* ... more colors ... */
}
```

### Products
Update product information in the HTML within the gallery section:
```html
<div class="product-card">
    <h3>Your Product Name</h3>
    <p>Your product description</p>
    <span class="price">$XX.XX</span>
</div>
```

### Contact Information
Update the contact details in the Contact section and Footer with your actual information.
